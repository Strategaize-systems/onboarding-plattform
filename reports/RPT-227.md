- `tailwind.config.ts.brand.primary = '#4454b8'` (Style Guide V2, hardcoded)
- `resolve.ts.STRATEGAIZE_DEFAULT_BRANDING.primaryColor = '#2563eb'` (arbitrary Tailwind-Blau)

- 20+ existierende Komponenten nutzen `bg-brand-primary` / `text-brand-primary` / `border-brand-primary` / `focus:ring-brand-primary` / `bg-gradient-to-r from-brand-primary-dark to-brand-primary` → alle aktuell #4454b8

2. **`sql/migrations/091a_align_strategaize_default_brand_color_styleguide_v2.sql`** (NEU, LIVE auf Hetzner 2026-05-12):

   - 3 GRANTs (authenticated + anon + service_role) re-applied.
   - Pre-Apply-Probe: 0 partner_branding_config-Rows mit altem `#2563eb` → kein Daten-UPDATE noetig.
   - Apply via base64-Pipe (sql-migration-hetzner.md): `NOTICE`-Output + `DO` + `CREATE FUNCTION` + 3 GRANTs erfolgreich.
