- **L-1 (Low, Test-Helper-Gotcha):** `withTestDb` aus `src/test/db.ts` wickelt alle Statements in EINE Tx (BEGIN/ROLLBACK). `created_at DEFAULT now()` ist Transaction-Start-Zeit, daher identisch fuer mehrere Inserts in derselben Tx. Sollte als IMP in Dev-System-Skill-Improvements aufgenommen werden.

Per Memory `feedback_async_always_coolify_cron.md` + Rule `coolify-test-setup.md` Z. 95-101 ist das ein bekanntes Pattern — Container-Name immer zur Laufzeit per `docker ps --format ...` resolven, nie hardcoden. Pattern wurde befolgt.
