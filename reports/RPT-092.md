## Methodik — Live-DB-Tunnel

Coolify-DB ist intern (10.0.3.3:5432, kein Host-Port-Mapping). Ad-hoc-SSH-Tunnel auf Container-IP:

```bash
ssh -fN -L 15432:10.0.3.3:5432 root@159.69.207.29
PG_PASS=$(ssh root@159.69.207.29 "docker exec supabase-db-bwkg80w04wgccos48gcws8cs-121729264357 printenv POSTGRES_PASSWORD")
export TEST_DATABASE_URL="postgresql://postgres:${PG_PASS}@127.0.0.1:15432/postgres"
npx vitest run --reporter=basic
```

Alle Test-Transactions laufen in BEGIN/ROLLBACK (siehe `src/test/db.ts:withTestDb`) — keine Daten-Persistenz, nur Audit-Spur via auth.users INSERT bleibt (Standard-Verhalten der Test-Harness, nicht V4-spezifisch).
