**Quelle:** Final-Check Build-Run dieser Session.

**Symptom:** `npm run build` schlaegt fehl mit `Error: supabaseUrl is required` beim Collecting Page Data fuer `/api/public/partner/[slug]`. **Mitigation post Go-Live:**
